## Reference 1 : Maquette Hero Burger

- **Screenshot** : `screenshots/ref1-hero.png`
- **Type** : Maquette / Fast-food
- **Ce qu'on retient** :
  - Composition hero : produit geant a droite, texte a gauche — impact visuel immediat
  - **Badges nutritionnels** (calories, graisses, proteines) — element cle pour StrictFood (positionnement healthy)
  - **Taille dominante du burger** — le produit est le hero, pas le texte
  - **Navbar** clean et bien structuree
- **A adapter** :
  - Transposer en dark mode (fond Charbon), remplacer les couleurs vives (vert/jaune) par la palette Or Braise

---

## Reference 2 : Maquette Homepage Pizza & Burger

- **Screenshot** : `screenshots/ref2-homepage.png`
- **Type** : Maquette / Fast-food Premium
- **Ce qu'on retient** :
  - **Typographies bold et expressives** — impact visuel fort, feeling premium
  - **Background multi-couleurs** : trainee de peinture orange + fond rouge, effet de texture dynamique
  - **Sections sous le hero** bien decoupees — rythme de page clair et structure
  - Produits eclaires sur fond sombre — coherent avec la DA "Dark Food Premium"
- **A adapter** :
  - Transposer l'effet background multi-couleurs sur un fond dark (Charbon) — utiliser des nuances chaudes (Or Braise) au lieu de l'orange/rouge
  - Garder le rythme sectionnel, adapter la densite visuelle (StrictFood = plus aere)
